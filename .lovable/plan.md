

## Target Operating Model – Interaktive Pyramide

Eine schicke, moderne interaktive Präsentation des Target Operating Model Frameworks als Pyramiden-Visualisierung mit dem bestehenden Farbschema (Slate/Teal-Palette).

### Seite: TOM Pyramide (Startseite)

**Header-Bereich**
- Moderner, minimalistischer Header ohne Brandname
- Titel "Target Operating Model" mit elegantem Subtitle
- Kurze Erklärung zur Interaktion

**Pyramiden-Visualisierung**
- 6 Ebenen als interaktive, klickbare Schichten (von oben nach unten breiter werdend)
- Farben gemäß bestehendem Schema: Teal (#64B5C6) als Akzent, dunkle Slate-Töne
- Hover-Effekte mit Scale, Glow und Accent-Linie
- Seitliche Pfeile "TOP-DOWN / DESIGN" und "BOTTOM-UP / UMSETZUNG"
- Staggered Entry-Animation beim Laden
- Schickerere Glassmorphism-Effekte und subtile Gradients für moderneren Look

**Detail-Modal (bei Klick auf eine Ebene)**
- Overlay mit Blur-Backdrop
- Tabs: Übersicht / Finance-Beispiel / Marketing-Beispiel
- Farblich abgestimmt auf die gewählte Ebene
- Smooth Ein-/Ausblend-Animationen

**Legende**
- Kompakte Legende am unteren Rand mit Erklärung zu Leserichtung (Design top-down, Umsetzung bottom-up)

**Design-Verbesserungen gegenüber Original**
- Tailwind CSS statt Inline-Styles für sauberen Code
- Modernere Typografie mit importierten Google Fonts
- Verfeinerte Glassmorphism/Frosted-Glass-Effekte
- Smoothere Animationen via Tailwind + CSS Keyframes
- Responsive Design für alle Bildschirmgrößen
- Dunkles Theme als Standard, passend zum Farbschema
